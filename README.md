# REAL ESTATE NARRATIVE ARCHITECT MICRO SAAS

Autonomous Self-Correcting Multi-Modal Agentic RAG Framework with Vision-to-Voice Synthesis, Global Market Intelligence & Advanced LLMOps.
This is an elite-level autonomous framework designed for the seamless synthesis of visual property data, real-time global market research, and high-fidelity voice-over orchestration.

## ✨ Key Capabilities
- **VISION-TO-VOICE SYNTHESIS**: End-to-end pipeline from 'property_image.jpg' analysis to final audio-ready Voice-over scripts.
- **TAVILY MARKET INTELLIGENCE**: Fetches live 2026 real estate trends and UNESCO cultural context for unparalleled narrative grounding.
- **AUTONOMOUS SELF-CORRECTION**: A multi-agent feedback loop where a Critic Node enforces strict Brand Identity Rules before approval.
- **HUMAN-IN-THE-LOOP (HITL)**: Executive confirmation stage for final masterpiece verification or iterative feedback.
- **ADVANCED LLMOPS**: Real-time token monitoring, cost tracking, and automated handling of API spikes (503 error mitigation).

## 🚀 Quick Start
1. **SETUP**: Run: `pip install -r requirements.txt`.
2. **CONFIGURATION**: Add your `GOOGLE_API_KEY` and `TAVILY_API_KEY` to the `.env` file.
3. **INPUTS**: Place `property_input.pdf` and `property_image.jpg` in the root folder.
4. **EXECUTION**: Run: `python main.py`.

## 📁 Project Structure
- `main.py`: Advanced LangGraph orchestration engine.
- `brand_identity_rules.txt`: Senior-level aesthetic & tone guidelines.
- `lessons_learned.txt`: Persistent memory for agent optimization.
- `evolution_log.txt`: Transparent trace of AI reasoning & edits.
- `FINAL_OUTPUT/`: Directory containing scripts, ads, and strategies.
